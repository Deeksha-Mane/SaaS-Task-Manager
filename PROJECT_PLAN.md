1️⃣ Project Overview
Name: SaaS Task Manager
Goal: A full-stack web app where authenticated users can manage personal tasks securely.

2️⃣ V1 Scope (Strict)
-User registration
-User login
-JWT authentication
-Create task
-Update task
-Delete task
-Mark complete
-Filter by status
-Pagination

3️⃣ Tech Stack

-Frontend:
React (Vite)
Axios

-Backend:
Node.js
Express

-Database:
MongoDB (Mongoose)

-Auth:
JWT
bcrypt

-Deployment:
Render (Backend) + Vercel (Frontend)
